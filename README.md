## Setup and Installation

Ensure that you have [Node.js](https://nodejs.org/en/download) v20+ installed on your machine.

First, install any dependencies required to run the project by running the following command
```
npm install
```

Afterwards, launch the website by running
```
npm run dev
```

Then, in a browser, navigate to
```
http://localhost:5173/
```
